# 🌤️ 오늘의 날씨 — 기상청 API 대시보드

> 공공데이터포털 기상청 단기예보 API를 활용한 실시간 날씨 웹 대시보드

**제출일:** 2026년 5월 10일  
**평가영역:** 데이터 활용 프로젝트 (API 웹 개발 공유)

---

## 📌 프로젝트 개요

본 프로젝트는 공공데이터포털(data.go.kr)에서 제공하는 **기상청 단기예보 API**를 활용하여 실시간 날씨 정보를 시각화하는 웹 대시보드를 개발한 결과물입니다.

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 오늘의 날씨 - 기상청 API 대시보드 |
| 활용 API | 기상청 단기예보 조회서비스 (공공데이터포털) |
| 개발 언어 | Python, HTML / CSS / JavaScript |
| 제출 파일 | `weather_dashboard.html`, `weather_api.py` |

---

## 🗂️ 활용 공공데이터 및 API 정보

- **데이터 출처:** 기상청 / 공공데이터포털 (https://www.data.go.kr)
- **API명:** 기상청_단기예보 ((구)동네예보) 조회서비스

### 사용 엔드포인트

| 엔드포인트 | 설명 |
|------------|------|
| `getUltraSrtNcst` | 초단기실황 — 현재 기온, 강수량, 풍속, 풍향, 습도 |
| `getVilageFcst` | 단기예보 — 오늘~모레까지 3시간 간격 예보 |

### 주요 응답 항목

| 카테고리 | 항목 | 단위 |
|----------|------|------|
| T1H | 기온 | °C |
| RN1 | 1시간 강수량 | mm |
| REH | 습도 | % |
| WSD | 풍속 | m/s |
| VEC | 풍향 | degree |
| PTY | 강수형태 (0=없음, 1=비, 2=비/눈, 3=눈, 4=소나기) | — |
| SKY | 하늘상태 (1=맑음, 3=구름많음, 4=흐림) | — |
| POP | 강수확률 | % |
| TMP | 예보 기온 | °C |

## 🛠️ 개발 내용

### `weather_dashboard.html` — 웹 대시보드

- 기상청 API를 브라우저에서 직접 호출하여 데이터 수신
- 지역 선택(서울, 부산, 대구, 인천, 광주, 대전, 강릉, 제주 등)에 따라 해당 격자 좌표(nx, ny)로 API 요청
- 현재 기온, 날씨 상태, 습도, 풍속, 강수량을 메인 카드에 표시
- 강수확률, 풍속, 습도, 강수량을 세부 지표 카드로 시각화
- 이후 12시간 시간대별 예보를 슬라이드 형식으로 제공
- 실시간 시계 및 날짜 표시 기능 포함
- 다크모드 디자인 적용으로 가독성 향상

### `weather_api.py` — Python 연동 스크립트

- `requests` 라이브러리를 이용해 기상청 API 호출
- 발표 기준 시각을 자동으로 계산하여 올바른 파라미터 생성
- 초단기실황 데이터로 현재 날씨 출력
- 단기예보 데이터로 이후 24시간 예보를 표 형식으로 출력
- 한국어 주석을 상세히 작성하여 가독성 확보

## 🖥️ 주요 기능 시연

### 기능 1 — 실시간 날씨 조회
지역 드롭다운에서 도시를 선택하고 **[조회]** 버튼을 클릭하면 현재 날씨가 즉시 표시됩니다.  
표시 항목: 기온(°C), 날씨 상태(맑음/흐림/비 등), 습도, 풍속, 강수량

### 기능 2 — 세부 기상 지표 카드
강수확률, 풍속, 습도, 1시간 강수량을 개별 카드로 표시합니다.  
풍속이 강할 경우(9m/s 이상) 카드 색상이 주황→빨간색으로 변해 위험 수준을 직관적으로 전달합니다.

### 기능 3 — 시간대별 예보 슬라이드
이후 12시간의 예보를 가로 스크롤 카드로 제공합니다.  
각 카드에는 날씨 아이콘, 기온, 강수확률이 표시됩니다.

### 기능 4 — 실시간 시계
화면 우측 상단에 현재 시각과 날짜를 실시간으로 표시합니다.

### 기능 5 — Python 스크립트 연동

```bash
pip install requests
python weather_api.py
```

출력 예시:
```
기온      : 18°C
하늘 상태  : 구름 많음
습도      : 65%
풍속      : 3.2 m/s
1시간 강수  : 0 mm
```

## 🔑 API 인증키 발급 절차

1. 공공데이터포털(data.go.kr) 회원가입 및 로그인
2. 검색창에 **「기상청 단기예보」** 입력 후 해당 API 검색
3. [활용신청] 클릭 → 활용목적 입력 → 신청 완료
4. 마이페이지 → 개발계정 → **인증키(Decoding)** 확인
5. `weather_dashboard.html` 열기 → [API 키 설정] 버튼 → 인증키 붙여넣기 → 저장
6. [조회] 버튼 클릭하여 정상 동작 확인

> ⚠️ 인증키는 신청 직후 발급되나, 실제 API 호출은 발급 후 약 1시간 이후부터 정상 작동합니다.
